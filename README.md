# Drawer
<div align=center><img width="150" height="150" src="./build/icons/icon_512x512.png"/></div>
<br/>

Drawer 是一个 macOS 上的历史剪贴板应用，基于 Electron + Vue

- 使用 <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>V</kbd> 呼出 Drawer 窗口
- 双击相应卡片进行复制
